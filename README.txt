All documentation is stored in the Documentation folder.

To run this project you must complete these steps:
  1) Import company.sql using phpmyadmin or another database tool
  2) Modify databaseInfo.php with appropriate login informtation.
  3) Run projectXSD.php to create the xml schema.
  4) run projectXML.php to create the xml data file.
  5) to run sample queries use queryTimes.php
  6) if you want to add more entries to the people table, modify the value in morePeople.php and run it.
